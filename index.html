Modelo de contenedor

El .container son etiquetas divs que sirven para maquetar nuestras paginas web, dentro de ellos puedes poner mas etiquetas en funcion las requieras a la hora de disenar tu pagina, existen varios modelos de contenedores desde pequeños hasta los mas grandes que abarcan la pantalla completa del dispositivo en funcion de la configuracion que se tenga.

Modelo de Malla de Bootstrap

Segun lo que he comprendido de lo visto en clases y investigaciones propias el modelo mala es un grid layout o una herramienta de maquetacion para nuestras paginas web el cual nos permite organizar nuestro contenido en filas y columnas para una vista mas presentable y agradable.
